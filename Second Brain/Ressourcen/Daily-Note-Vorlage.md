# Daily Note Vorlage

*Kopiere diese Vorlage in Templater oder Daily Notes Plugin.*
*Ablage: `Selbstentwicklung/Tagesreflexion/`*

---

```markdown
# {{date:DD.MM.YYYY}} – {{date:dddd}}

## 🎯 Heute schaffe ich
- 
- 
- 

## 💼 Ausbildung / Schule
- 

## 💡 Ideen & Gedanken
- 

## 📖 Gelernt heute
- 

## 🔁 Reflexion (abends)

**Was lief gut?**

**Was lief nicht gut?**

**Was nehme ich mit?**

---
*Energie heute (1-10):* 
*Stimmung heute (1-10):*
```

---

## Obsidian-Setup für Daily Notes

1. **Plugin aktivieren:** Einstellungen → Core plugins → Daily notes ✅
2. **Ablage festlegen:** Einstellungen → Daily notes → New file location → `Selbstentwicklung/Tagesreflexion`
3. **Templater installieren:** Community plugins → Templater → Vorlage auf diese Datei zeigen lassen
4. **Hotkey:** Strg+Shift+D für neue Daily Note (empfohlen)

---

*Einfach halten. 5 Minuten morgens, 5 Minuten abends. Mehr braucht es nicht.*
