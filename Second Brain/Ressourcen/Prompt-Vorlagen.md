# 💬 Prompt-Vorlagen

*Häufige Anfragen als fertige Vorlagen. Einfach kopieren, Klammern ausfüllen, absenden.*

---

## 🎓 Schule & Lernen

**Erklärung:**
```
Ich bin Azubi KfB im 1. Jahr. Erkläre mir [Thema] einfach mit einem Praxis-Beispiel. Max. 5 Sätze.
```

**Lernzusammenfassung aus Mitschrift:**
```
Hier ist meine Mitschrift: [Text einfügen]
Erstell daraus eine strukturierte Lernzusammenfassung für die Wiederholung.
```

**Prüfungsvorbereitung:**
```
Erstell mir 15 Prüfungsfragen mit Antworten zum Thema [Thema] für eine KfB-Prüfung im 1. Jahr.
```

**Lernplan:**
```
Ich habe bis [Datum] Zeit und muss folgende Themen lernen: [Liste].
Ich habe täglich [X] Minuten. Erstell mir einen Lernplan.
```

---

## 📅 Planung & Organisation

**Wochenplan:**
```
Hier ist was ich diese Woche erreichen will: [Liste]
Ich arbeite Mo/Do/Fr (8–16 Uhr), Schule Di/Mi (8–13:10 Uhr).
Erstell mir einen realistischen Wochenplan mit Zeitblöcken.
```

**Entscheidung:**
```
Ich bin unentschlossen zwischen [Option A] und [Option B].
Kontext: [kurze Beschreibung].
Erstell mir eine ehrliche Pro/Contra-Analyse und sag mir deine Empfehlung.
```

---

## 💼 Ausbildung / Arbeit

**E-Mail schreiben:**
```
Schreib mir eine professionelle aber nicht zu steife E-Mail an [Empfänger / Ausbilder / Lehrer].
Ich will mitteilen: [Kernaussage]. Ich bin Azubi im 1. Jahr KfB.
```

## 🚀 Business

**Idee analysieren:**
```
Ich habe folgende Business-Idee: [Beschreibung]
Analysiere sie kritisch: Stärken, Risiken, Zielgruppe,
wie ich sie mit wenig Aufwand und ohne Geldrisiko testen kann.
```

**Marktrecherche:**
```
Ich will mehr über [Markt/Nische] wissen.
Was sind aktuelle Trends, wer sind typische Kunden, was machen erfolgreiche Anbieter?
Fass die wichtigsten Punkte für mich zusammen.
```

## 🧠 Reflexion & Selbstentwicklung

**Abend-Reflexion:**
```
Hier ist meine heutige Reflexion: [Text aus Daily Note]
Was sind die wichtigsten Learnings? Was sollte ich morgen anders machen?
```

**Wochen-Review:**
```
Hier ist meine Woche kurz zusammengefasst: [Text]
Was sind die wichtigsten Learnings? Bin ich meinen Zielen näher gekommen?
Was nehme ich für nächste Woche mit?
```

---

## 🤖 Wiki-Pflege

**Neue Quelle einpflegen:**
```
Ich habe eine neue Quelle für das Wiki. Hier ist der Inhalt: [Text/Link]
Bitte: 1) Wichtigste Erkenntnisse zusammenfassen, 2) Passende Seite im Wiki anlegen,
3) Querverweise zu bestehenden Seiten setzen, 4) index.md und log.md aktualisieren.
```

**Wiki-Check:**
```
Mach einen Lint-Durchgang über mein Wiki.
Suche nach: Widersprüchen, veralteten Infos, Seiten ohne Querverweise,
fehlenden Seiten für erwähnte Konzepte, Lücken die ich füllen sollte.
```
