## Installationsweg

1. Ordner aus diesem Verzeichnis nehmen
2. Auf deinem PC kopieren nach: `[dein Cowork-Ordner]/.claude/skills/[skill-name]/`
3. Cowork neu starten
4. Skill ist aktiv
