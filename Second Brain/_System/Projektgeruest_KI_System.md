## 1️⃣ Claude & KI-Tools einrichten

### Ziel
Claude Pro + Google One Premium voll ausschöpfen. Jeden Tag sinnvoll nutzen.

### Aufgaben
- [ ] Memory in Einstellungen aktivieren (Einstellungen → Capabilities)
- [ ] 5 Projects anlegen: Ausbildung, Business, Selbstentwicklung, Kreatives, Wochenpläne
- [ ] In jedem Project: kurze Beschreibung hinterlegen wer ich bin + worum es geht
- [ ] Prompt-Cheatsheet erstellen (häufige Anfragen als Vorlagen)
- [ ] NotebookLM: erstes Notizbuch für Berufsschule anlegen (notebooklm.google.com)
- [ ] Gemini in Gmail aktivieren und einmal testen
- [ ] Perplexity als Google-Ersatz testen (perplexity.ai)

### Wie ich die Tools täglich nutze

| Tool | Wann | Wofür |
|---|---|---|
| Claude Pro | Täglich | Denken, Planen, Texte, Lernen, Reflexion |
| NotebookLM | Schultage (Di, Mi) | Lernmaterial als Podcast, Zusammenfassungen |
| Gemini Gmail | Arbeitstage (Mo, Do, Fr) | E-Mails schreiben, zusammenfassen |
| Perplexity | Bei Recherche-Bedarf | Aktuelle Infos, schnelle Fakten |

## 2️⃣ Obsidian Second Brain aufbauen

### Ziel
Alle Gedanken, Learnings, Ideen und Pläne an einem Ort. KI kann darauf zugreifen.

### Ordnerstruktur

```
📁 Second Brain
├── 📁 Inbox           → Alles landet erstmal hier
├── 📁 Ausbildung
│   ├── 📁 Berufsschule   → Mitschriften, Zusammenfassungen
│   └── 📁 NX Logistics   → Aufgaben, Notizen aus der Arbeit
├── 📁 Business
│   ├── 📁 Ideen          → Rohe Business-Ideen
│   ├── 📁 Recherche      → Markt, Konkurrenz, Trends
│   └── 📁 Aktionen       → Was ich konkret tue
├── 📁 Selbstentwicklung
│   ├── 📁 Ziele          → Jahres-, Monats-, Wochenziele
│   ├── 📁 Bücher & Artikel → Learnings aus dem was ich lese/höre
│   └── 📁 Tagesreflexion → Daily Notes
└── 📁 Ressourcen      → Vorlagen, Prompts, hilfreiche Links
```

### Aufgaben
- [ ] Obsidian herunterladen & installieren (obsidian.md)
- [ ] Vault anlegen mit oben genannter Struktur
- [ ] Plugin installieren: **Daily Notes** (täglich automatisch neue Notiz)
- [ ] Plugin installieren: **Templater** (Vorlagen für Daily Note)
- [ ] Plugin installieren: **Smart Connections** (KI-Chat mit eigenen Notizen)
- [ ] Plugin installieren: **Dataview** (Notizen wie Datenbank abfragen)
- [ ] Erste Daily Note Vorlage anlegen (siehe Abschnitt Routinen)
- [ ] Obsidian Sync einrichten ODER Obsidian Git Plugin (kostenlose Alternative)

### Daily Note Vorlage

## 3️⃣ Routinen & Gewohnheiten

### Ziel
Feste Momente im Tag für KI-Nutzung und Reflexion etablieren.

### Morgen-Routine (5-10 Min)
1. Daily Note öffnen in Obsidian
2. 3 Tages-Prioritäten aufschreiben
3. Falls nötig: kurze Frage an Claude zur Planung

### Abend-Routine (5-10 Min)
1. Daily Note: Reflexion ausfüllen (Was lief gut / schlecht / Learnings)
2. Morgen kurz vorbereiten

### Wochen-Review (Sonntag, 15-20 Min)
1. Wochenrückblick mit Claude: „Hier ist meine Woche [kurz zusammenfassen]. Was sind die wichtigsten Learnings?"
2. Nächste Woche planen: Was muss passieren?
3. Ziele checken: Bin ich auf Kurs?

### Mein Wochenrhythmus

| Tag | Wo | Zeit | KI-Nutzung |
|---|---|---|---|
| Montag | NX Logistics | 8-16 Uhr | Morgens planen, abends reflektieren |
| Dienstag | Berufsschule | 8-13:10 Uhr | NotebookLM zum Lernen |
| Mittwoch | Berufsschule | 8-13:10 Uhr | NotebookLM zum Lernen |
| Donnerstag | NX Logistics | 8-16 Uhr | Morgens planen, abends reflektieren |
| Freitag | NX Logistics | 8-16 Uhr | Woche abschließen |
| Samstag | Frei | - | Business-Zeit, Ideen entwickeln |
| Sonntag | Frei | - | Wochen-Review & neue Woche planen |

### Aufgaben
- [ ] Daily Note Vorlage in Obsidian anlegen
- [ ] Feste Zeiten für Morgen- und Abend-Routine im Kalender blocken
- [ ] Wochen-Review als wiederkehrendes Event anlegen (Sonntag, 19 Uhr)
- [ ] Ersten Wochen-Review mit Claude machen

## 4️⃣ Business-Aufbau

### Ziel
Schrittweise eine eigene Business-Idee entwickeln, testen und starten – parallel zur Ausbildung.

### Phase 1 – Ideen finden (Monat 1-2)
- [ ] Mit Claude: Brainstorming welche Business-Ideen zu mir passen
- [ ] Interessen, Fähigkeiten und Ressourcen auflisten
- [ ] 3-5 Ideen auswählen und tiefer analysieren
- [ ] Erste Marktrecherche mit Perplexity + Claude

### Phase 2 – Idee testen (Monat 2-4)
- [ ] Eine Idee als MVP (kleinstmögliche Version) definieren
- [ ] Erste Schritte ohne Geldrisiko testen
- [ ] Feedback einholen

### Phase 3 – Aufbauen (ab Monat 4)
- [ ] Erste echte Schritte in Richtung Umsatz
- [ ] Content-Strategie mit KI entwickeln
- [ ] Erste Automatisierungen mit Make.com

### Ressourcen
- Business-Ideen Brainstorming: Claude (Project „Business")
- Marktrecherche: Perplexity AI
- Content-Erstellung: Claude + Whisk/Flow (Google)
- Automatisierung: Make.com (kostenloser Einstieg)

## 5️⃣ Schule & Ausbildung optimieren

### Ziel
Berufsschule und Ausbildung effizienter meistern – weniger Stress, bessere Ergebnisse.

### Für die Berufsschule (Di, Mi)
- [ ] NotebookLM Notizbuch „Berufsschule" anlegen
- [ ] Nach jedem Schultag: Mitschriften hochladen oder fotografieren
- [ ] Audio Overview aus dem Stoff generieren → beim Heimweg anhören
- [ ] Vor Prüfungen: Lernguide + Prüfungsfragen generieren lassen

### Für die Ausbildung (Mo, Do, Fr)
- [ ] Aufgaben und Projekte in Obsidian dokumentieren
- [ ] Bei Unklarheiten: Claude fragen (z.B. Buchführung, Büromanagement-Themen)
- [ ] Wichtige E-Mails mit Gemini in Gmail schreiben

### KI-Prompt-Vorlagen für die Schule

**Erklärung:**
> „Ich bin Azubi KfB im 1. Jahr. Erkläre mir [Thema] einfach mit einem Praxis-Beispiel. Max. 5 Sätze."

**Prüfungsvorbereitung:**
> „Erstell mir 15 Prüfungsfragen mit Antworten zum Thema [Thema] für eine KfB-Prüfung."

**Lernplan:**
> „Ich habe bis [Datum] Zeit und muss folgende Themen lernen: [Liste]. Ich habe täglich [X] Minuten. Erstell mir einen Lernplan."

**Zusammenfassung:**
> „Hier ist meine Mitschrift: [Text]. Erstell daraus eine strukturierte Zusammenfassung für die Wiederholung."
